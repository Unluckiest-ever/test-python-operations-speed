Test Python Operation Speed
===========
### Short description
Window application for speed test of Python and Numpy arithmetic operations (+, -, /, *, ^).  

User can input:
* test count
* min number
* max number
* test run count
* operation:
  * addition
  * subtraction
  * multiplication
  * division
  * pow
* number type
***
